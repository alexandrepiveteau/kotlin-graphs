randomWalkWeighted
Link copied to clipboard
inline fun <G : SuccessorsWeight> G.randomWalkWeighted(from: Vertex, random: Random = Random, action: (Vertex) -> Unit)

Performs a random walk on the graph, starting from the given from vertex, and performs the given action on each vertex. The walk may be infinite if the graph contains cycles, and the walk will stop if the vertex has no successors. The walk will be weighted, and the probability of choosing a successor is proportional to the weight of the link. Negative weights are not supported, and will be ignored and treated as if they were 0.
